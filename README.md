# SAI PHOTOGRAPHY Website

Welcome to the premium cinematic photography website for SAI PHOTOGRAPHY.

## Features
- **Modern Dark Luxury Theme**: Custom-built with a palette of deep black and rich gold.
- **Cinematic Experience**: Smooth transitions, reveal animations, and image hover zoom effects.
- **Fully Responsive**: Mobile-first design that adapts beautifully to any device.
- **High Performance**: Vanilla HTML, CSS, and JS for maximum speed and SEO optimization.
- **Lead Generation**: Strategically placed "Book Now" and contact forms.

## Pages
- `index.html`: Home page with hero section and service highlights.
- `about.html`: Detailed creator profile for Sundaramoorthy.
- `services.html`: Extensive list of services including wedding, baby, and event shoots.
- `gallery.html`: Filterable portfolio with a built-in lightbox preview.
- `pricing.html`: Premium pricing cards for various packages.
- `contact.html`: Multi-channel contact options (form, WhatsApp, social links).

## Assets used
- **Fonts**: Playfair Display (Luxury Headers), Inter (Clean Body Text).
- **Icons**: Font Awesome 6.4.0.
- **Images**: High-quality placeholders from Unsplash (Optimized for photography themes).

## Implementation
1. Simply open `index.html` in any browser to view the site.
2. Custom scripts are in `script.js`.
3. Global styles are in `style.css`.
